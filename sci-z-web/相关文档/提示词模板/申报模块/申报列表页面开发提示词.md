# 申报列表页面开发提示词

## 📋 页面信息
- **原型图路径**：`原型图/申报模块/申报列表.html`
- **页面类型**：列表页面
- **页面名称**：申报管理列表
- **主要功能**：申报数据展示、搜索筛选、状态管理、操作处理
- **对应文件路径**：`src/views/Declaration/List.vue`

## 🎨 原型图视觉分析

### **整体布局**
- 背景色：`#f7f9fc`
- 容器：`padding: 20px`，`min-height: calc(100vh - 56px)`
- 页面结构：页面标题 + 搜索区域 + 表格区域 + 分页

### **页面头部**
- 标题：`申报管理`，字体大小24px，颜色`#1e3a8a`
- 新建按钮：主要按钮，带加号图标，触发新建申报

### **搜索筛选区域**
- 白色卡片背景，圆角8px，阴影效果
- 关键词搜索：输入框，支持回车搜索
- 状态筛选：下拉选择器，包含全部、申报中、申报成功、申报失败
- 操作按钮：搜索按钮、重置按钮

### **表格区域**
- 使用Element Plus表格组件，不是原生HTML表格
- 列定义：申报编号、申报人、申报时间、研究方向、研究领域、研究课题、状态、操作
- 状态列：可点击下拉修改状态
- 操作列：查看详情、下载（成功状态）、预览（成功状态）

const handleStatusEdit = async (id, newStatus) => {
  const statusLabels = {
    submitting: t('declaration.statusSubmitting'),
    success: t('declaration.statusSuccess'),
    failed: t('declaration.statusFailed')
  }
  
  const declaration = declarations.value.find(d => d.id === id)
  if (!declaration) return
  
  const oldStatus = declaration.statusType
  const oldStatusLabel = declaration.status
  
  if (newStatus === oldStatus) return
  
  try {
    await ElMessageBox.confirm(
      t('declaration.confirmStatusChange', { status: statusLabels[newStatus] }),
      t('declaration.confirmTitle'),
      {
        confirmButtonText: t('common.confirm'),
        cancelButtonText: t('common.cancel'),
        type: 'warning',
        customClass: 'modern-confirm-dialog'

    } else {
      // 接口错误
      logger.error('Status update failed', error)
      ElMessage.error(t('declaration.statusUpdateFailed'))
    }
  }
}

  loadDeclarations()
