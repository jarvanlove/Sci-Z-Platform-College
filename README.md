# Sci-Z Platform College

## 项目简介

高校科研项目管理平台 - 专为高校科研项目管理设计的综合性平台系统。

## 项目结构

```
Sci-Z-Platform-College/
├── sci-z-server/          # 后端服务
├── sci-z-web/            # 前端项目
│   ├── 原型图/           # UI原型设计
│   ├── 开发计划.md       # 开发计划
│   └── 高校科研项目管理平台需求文档.md
└── 流程图/               # 架构设计图
```

## 功能模块

- 🔐 认证模块（登录、注册、密码重置）
- 📊 仪表板
- 👤 用户中心
- 📝 申报模块
- 📁 项目模块
- ✅ 验收模块
- 🤖 AI 助手
- ⚙️ 系统管理

## 技术栈

### 后端

- Java Spring Boot
- MySQL
- Redis

### 前端

- Vue.js
- Element UI
- Axios
